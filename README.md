# 🎉 Birthday Surprise Website

This is a creative and interactive birthday website built using HTML, CSS, and JavaScript to celebrate a special person.

## 🌟 Project Description

This website is designed as a digital birthday surprise. It includes animations, music, gallery, and personalized wishes to make the experience engaging and memorable.

## 🚀 Features

- 🎈 Animated birthday theme with balloons & confetti
- 🎵 Background music with play/pause control
- 🎁 Surprise button interaction
- 📸 Image gallery (Memory section)
- 💌 Birthday wishes cards
- ✍️ Send a wish feature (user input)
- 🧑‍💻 Personalized "About" section

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Canvas Confetti Library
